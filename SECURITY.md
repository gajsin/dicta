# Политика безопасности

Не публикуйте уязвимости и реальные API-ключи в Issues.

Для приватного сообщения используйте раздел Security Advisories репозитория GitHub. Укажите версию Dicta, шаги воспроизведения, возможное влияние и безопасный пример без пользовательских данных.

Поддерживается последняя опубликованная версия. Исправления безопасности выпускаются по мере готовности.

## Обработка данных

- API-ключи хранятся локально в `%APPDATA%\Dicta\settings.json` по явному выбору проекта.
- Не публикуйте `settings.json`, не прикладывайте его к Issues и не добавляйте в репозиторий.
- История диктовок хранится локально в WebView-хранилище и может быть отключена.
- Аудио удерживается в памяти только для повторной отправки после ошибки и удаляется после успешной обработки или новой записи.
